# 🚀 Personal Portfolio Website

A modern and responsive **Personal Portfolio Website** built using **Spring Boot**, **Thymeleaf**, HTML, CSS, and JavaScript.

🌐 **Live Demo:**
👉 https://portfolio2-d1wy.onrender.com

---

## 📌 Features

* 🏠 Home / Landing Page
* 👨‍💼 About Section
* 💼 Projects Showcase
* 🏆 Achievements
* 📞 Contact Section
* ⚡ Fast and responsive UI
* 🌙 Clean and modern design

---

## 🛠️ Tech Stack

* **Backend:** Java, Spring Boot
* **Frontend:** HTML, CSS, JavaScript
* **Template Engine:** Thymeleaf
* **Build Tool:** Maven
* **Deployment:** Render

## ⚙️ Installation & Run Locally

### 1️⃣ Clone the repository

```
git clone https://github.com/your-username/Portfolio1.git
cd Portfolio1
```

### 2️⃣ Build the project

```
./mvnw clean install
```

### 3️⃣ Run the application

```
java -jar target/*.jar
```

### 4️⃣ Open in browser

```
http://localhost:8080
```

---

## 🐳 Docker Setup

```
docker build -t portfolio-app .
docker run -p 8080:8080 portfolio-app
```
